#DUXCMS 2.0

DUXCMS是一款针对于中小企业、新闻站点等使用的建站程序

2.0版本完全基于THINKPHP重写

环境支持：
php5.3+
mysql5.1

浏览器支持：
ie9+ chrome firefox 
因特性问题暂不支持IE9以下浏览器，后期根据情况做IE8兼容性

----

#说明

默认后台帐号密码为：admin
默认后台地址为：http://域名/index.php?m=admin
